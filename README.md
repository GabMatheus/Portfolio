## Como Usar

1. Clone o repositório:
    ```bash
    git clone https://github.com/GabMatheus/portfolio.git
    ```

2. Navegue até o diretório do projeto:
    ```bash
    cd portfolio
    ```

3. Abra o arquivo `index.html` no seu navegador ou use um servidor local para visualizá-lo.
