Por trabalhar em uma empresa privada, grande parte dos projetos que desenvolvi é destinada exclusivamente para o uso interno, não podendo ser divulgada publicamente.

No entanto, posso compartilhar, além deste portfólio desenvolvido por mim, algumas pequenas automações que criei para melhorar atividades rotineiras da minha empresa. Elas estão disponíveis, explicadas e codadas no meu GitHub:

Gerador de Códigos de Barras: Este projeto de automação foi desenvolvido para gerar códigos de barras para produtos. Ele surgiu da necessidade de resolver o problema de códigos ilegíveis ou de difícil acesso para os funcionários. Com ele, é possível criar uma cópia acessível, facilitando a entrega de informações aos clientes.

Consulta de CNPJ: Desenvolvi este projeto para consultar CNPJs na API da Receita Federal e retornar dados úteis. Ele extrai informações de empresas e gera um arquivo TXT que é importado no ERP Dominio Sistemas, automatizando o cadastro de clientes e fornecedores na área de contabilidade.

Distância entre Cidades: Este aplicativo consulta a API do Google Maps para calcular a distância entre cidades no Brasil. Ele pode mostrar a distância direta entre duas cidades ou de uma cidade para várias outras registradas na API.

Renomeador de PDFs: Este script em Python renomeia arquivos PDF de comprovantes da empresa, que originalmente tinham nomes confusos. Ele extrai o CNPJ do documento e renomeia o arquivo de forma prática, facilitando a consulta sem precisar abrir cada arquivo.
