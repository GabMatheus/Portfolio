Gerador de Códigos de Barras: Este projeto de automação foi desenvolvido para gerar códigos de barras para produtos. Ele surgiu da necessidade de resolver o problema de códigos ilegíveis ou de difícil acesso para os funcionários. O funcionamento é simples, o usuário digita os códigos e ao finalizar o programa cria no desktop um docx com todos os códigos para impressão

Consulta de CNPJ: Desenvolvi este projeto para consultar CNPJs com a API da Receita Federal e retornar dados úteis para nosso setor de contabilidade. Ele extrai informações de empresas e gera um arquivo TXT que é importado posteriormente no ERP Dominio Sistemas, automatizando o cadastro de clientes e fornecedores na área de contabilidade.

Distância entre Cidades: Este aplicativo consulta a API do Google Maps para calcular a distância entre cidades no Brasil. Ele pode mostrar a distância direta entre duas cidades ou de uma cidade para várias outras registradas na API.

Site nutricional (acesso ao GIT): Este Site foi criado para a atividade extensionista do curso de Análise e desenvolvimento de sistemas e seu link é https://gabmatheus.github.io/Site_nutri/. Nele temos algumas informações sobre alimentos, possui uma calculadora de macronutrientes e tem também o contato de um nutricionista parceiro para quem se interessar pelo tema de saúde nutricional.

Soldiers vs Zombies: Aqui temos um jogo arcade 2D que desenvolvi em python usando a biblioteca Pygame e salvando os scores num BD SQLite3 para a matéria de Linguagem de Programação Aplicada do meu curso da faculdade.

Consultar Produtos/Cartões: Este projeto é uma aplicação desktop que permite a consulta de informações sobre produtos e cartões em aberto para a empresa. Desenvolvida utilizando Electron no frontend e Flask servindo a API no backend, a aplicação acessa dados armazenados em no banco de dados PostgreSQL. A interface gráfica foi construída com HTML, CSS e JavaScript.
